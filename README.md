## Getting Started

1. Clone the repo:

```console
$ git clone https://github.com/m-ikiara/plum8ob.git
```

2. Navigate into the new directory and execute the following:

```console
$ make
```

3. Walk through the code in `src/llvm` and the binaries in `bin/`.
